# Pinhalense auxílio de pesquisa e automatização – Estudo de Caso

## RNF01 — Disponibilidade
 O sistema deve estar disponível na maior parte do tempo, momentos de manutenção programada.

## RNF02 — Segurança
 Todos os dados sensíveis serão armazenados de forma criptografada.

## RNF03 — Usabilidade
 A interface deve ser intuitiva e responsiva.
